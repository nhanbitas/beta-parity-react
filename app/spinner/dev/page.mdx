| -------------------------------------------------- | ------------------ | ----------------- |

## Spinner API Reference

### `<Spinner />`

| **Prop**         | **Type**                                | **Default**  | **Description**                               |
| ---------------- | --------------------------------------- | ------------ | --------------------------------------------- |
| **variant**      | 'circular' \| 'dotted' \| 'sunburst'    | _'circular'_ | Spinner style variant                         |
| **color**        | 'neutral' \| 'accent'                   | _'neutral'_  | Spinner color                                 |
| **size**         | 'sm' \| 'md' \| 'lg' \| 'xl'            | _'md'_       | Spinner size                                  |
| **...spanProps** | `React.HTMLAttributes<HTMLSpanElement>` |              | All native span props (className, style, ...) |

> **_Note:_** _Spinner_ **inherits** all native props from `HTMLSpanElement`.
