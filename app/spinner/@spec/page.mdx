## üß© Spinner API Reference

### `<Spinner />`

| **Prop**        | **Type**             | **Default** | **Description**                              |
| --------------- | -------------------- | ----------- | -------------------------------------------- |
| **size**        | 'sm' \| 'md' \| 'lg' | _'md'_      | Spinner size                                 |
| **color**       | string               |             | Spinner color                                |
| **thickness**   | number               |             | Stroke thickness                             |
| **...svgProps** | SVGProps             |             | All native SVG props (className, style, ...) |

> **_Note:_** _Spinner_ **inherits** all native props from **SVGElement**.
