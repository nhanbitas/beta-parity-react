## üß© Tooltip API Reference

### `<Tooltip />`

| **Prop**            | **Type**             | **Default** | **Description**                              |
| ------------------- | -------------------- | ----------- | -------------------------------------------- |
| **content**         | string \| ReactNode  |             | Tooltip content                              |
| **placement**       | string               | _'top'_     | Tooltip placement                            |
| **visible**         | boolean              |             | Controlled visibility                        |
| **defaultVisible**  | boolean              | _false_     | Default visibility (uncontrolled)            |
| **onVisibleChange** | function (visible)   |             | Callback when visibility changes             |
| **trigger**         | string               | _'hover'_   | Trigger type (hover, click, focus)           |
| **...divProps**     | HTMLDivElement props |             | All native div props (className, style, ...) |

> **_Note:_** _Tooltip_ **inherits** all native props from **HTMLDivElement**.
