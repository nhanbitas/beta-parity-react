## Tooltip API Reference

### `<Tooltip />`

| **Prop**           | **Type**             | **Default** | **Description**                              |
| ------------------ | -------------------- | ----------- | -------------------------------------------- |
| **content**        | string \| ReactNode  |             | Tooltip content                              |
| **position**       | string               | _'top'_     | Tooltip placement                            |
| **isToggle**       | boolean              | _false_     | Toggle on click                              |
| **delay**          | number               | _0_         | Delay before show on hover (ms)              |
| **touchDelay**     | number               |             | Delay before show on touch (ms)              |
| **isSafePolygon**  | boolean              | _false_     | Safe polygon interaction                     |
| **controlledOpen** | boolean              |             | Controlled visibility                        |
| **...divProps**    | HTMLDivElement props |             | All native div props (className, style, ...) |

> **_Note:_** _Tooltip_ **inherits** all native props from **HTMLDivElement**.
