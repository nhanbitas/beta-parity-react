## üß© Pagination API Reference

### `<Pagination />`

| **Prop**            | **Type**             | **Default** | **Description**                              |
| ------------------- | -------------------- | ----------- | -------------------------------------------- |
| **page**            | number               |             | Current page                                 |
| **pageSize**        | number               | _10_        | Items per page                               |
| **total**           | number               |             | Total items                                  |
| **onChange**        | function (page)      |             | Callback when page changes                   |
| **showSizeChanger** | boolean              | _false_     | Show page size changer                       |
| **sizeOptions**     | number[]             |             | Page size options                            |
| **...divProps**     | HTMLDivElement props |             | All native div props (className, style, ...) |

> **_Note:_** _Pagination_ **inherits** all native props from **HTMLDivElement**.
