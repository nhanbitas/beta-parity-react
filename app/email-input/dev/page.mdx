## EmailInput API Reference

### `<EmailInput />`

| **Prop**           | **Type**                       | **Default** | **Description**                           |
| ------------------ | ------------------------------ | ----------- | ----------------------------------------- |
| **domain**         | string \| string[]             |             | Domain or list of domains for suggestions |
| **onDomainChange** | function (domain)              |             | Callback when domain changes              |
| **domainValue**    | string                         |             | Currently selected domain (controlled)    |
| **...InputProps**  | [See Input](../base-input/dev) |             | Inherits all props from Input             |

> _Note:_ _EmailInput_ inherits all props from Input and the native input element. See [Input documentation](../base-input/dev) for full details on available props and tokens.
