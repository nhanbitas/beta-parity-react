## EmailInput API Reference

### `<EmailInput />`

| **Prop**           | **Type**                  | **Default** | **Description**                            |
| ------------------ | ------------------------- | ----------- | ------------------------------------------ |
| **type**           | string                    | _'email'_   | Input type (always 'email' for EmailInput) |
| **domain**         | string \| string[]        |             | Domain or list of domains for suggestions  |
| **onDomainChange** | function (domain)         |             | Callback when domain changes               |
| **domainValue**    | string                    |             | Currently selected domain (controlled)     |
| **...InputProps**  | [See Input](../BaseInput) |             | Inherits all props from Input              |

> _Note:_ _EmailInput_ inherits all props from Input and the native input element. See [Input documentation](../BaseInput) for full details on available props and tokens.
