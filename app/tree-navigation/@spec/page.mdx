## TreeNavigation API Reference

### `<TreeNavigation />`

| **Prop**        | **Type**             | **Default** | **Description**                                 |
| --------------- | -------------------- | ----------- | ----------------------------------------------- |
| **items**       | array                |             | Tree data structure (array of navigation items) |
| **...divProps** | HTMLDivElement props |             | All native div props (className, style, ...)    |

> **_Note:_** _TreeNavigation_ **inherits** all native props from **HTMLDivElement**.

## Keyboard behaviors

- All navigation items are focusable via Tab.
- Arrow keys (Up/Down) move focus between items (see accessibility hook in implementation).
- Enter/Space activates the focused item.
- ARIA attributes (`aria-label`, `aria-selected`, `aria-expanded`) are set for accessibility.
