## üß© TreeNavigation API Reference

### `<TreeNavigation />`

| **Prop**        | **Type**             | **Default** | **Description**                              |
| --------------- | -------------------- | ----------- | -------------------------------------------- |
| **data**        | array                |             | Tree data structure                          |
| **onSelect**    | function (key)       |             | Callback when node is selected               |
| **selectedKey** | string \| number     |             | Currently selected key                       |
| **expandAll**   | boolean              | _false_     | Expand all nodes by default                  |
| **renderNode**  | function (node)      |             | Custom node renderer                         |
| **...divProps** | HTMLDivElement props |             | All native div props (className, style, ...) |

> **_Note:_** _TreeNavigation_ **inherits** all native props from **HTMLDivElement**.
