| **Token**                                  | **Color for**                | **State/Meaning** | **CSS Selector(s)**                                                          |
| ------------------------------------------ | ---------------------------- | ----------------- | ---------------------------------------------------------------------------- |
| **--par-color-bg-field-enabled**           | Input background             | Enabled           | `.number-input .par-input:enabled`                                           |
| **--par-color-bg-field-hovered**           | Input background             | Hovered           | `.number-input .par-input:hover`                                             |
| **--par-color-bg-field-disabled**          | Input background             | Disabled          | `.number-input .par-input:disabled`                                          |
| **--par-color-bg-field-readonly**          | Input background             | Readonly          | `.number-input .par-input[readonly]`                                         |
| **--par-color-bg-field-invalid**           | Input background             | Invalid/Error     | `.number-input .par-input.error-state`                                       |
| **--par-color-border-field-enabled**       | Input border                 | Enabled           | `.number-input .par-input:enabled`                                           |
| **--par-color-border-field-hovered**       | Input border                 | Hovered           | `.number-input .par-input:hover`                                             |
| **--par-color-border-field-disabled**      | Input border                 | Disabled          | `.number-input .par-input:disabled`                                          |
| **--par-color-border-field-invalid**       | Input border                 | Invalid/Error     | `.number-input .par-input.error-state`                                       |
| **--par-color-border-field-focused**       | Input border                 | Focused           | `.number-input .par-input:focus`                                             |
| **--par-color-text-field-placeholder**     | Placeholder text             | Any               | `.number-input .par-input::placeholder`                                      |
| **--par-color-text-field-filled**          | Input text                   | Filled            | `.number-input .par-input`                                                   |
| **--par-color-text-field-label**           | Label text                   | Any               | `.number-input-label`                                                        |
| **--par-color-text-field-disabled**        | Input text                   | Disabled          | `.number-input .par-input:disabled`                                          |
| **--par-color-text-field-readonly**        | Input text                   | Readonly          | `.number-input .par-input[readonly]`                                         |
| **--par-color-text-field-invalid**         | Input text                   | Invalid/Error     | `.number-input .par-input.error-state`                                       |
| **--par-color-icon-field-button-enabled**  | Icon color (button enabled)  | Enabled           | `.number-input .clear-button:enabled`, `.number-input .input-icon:enabled`   |
| **--par-color-icon-field-button-hovered**  | Icon color (button hovered)  | Hovered           | `.number-input .clear-button:hover`, `.number-input .input-icon:hover`       |
| **--par-color-icon-field-button-pressed**  | Icon color (button pressed)  | Pressed           | `.number-input .clear-button:active`, `.number-input .input-icon:active`     |
| **--par-color-icon-field-button-disabled** | Icon color (button disabled) | Disabled          | `.number-input .clear-button:disabled`, `.number-input .input-icon:disabled` |

> _Note: NumberInput uses the same design tokens as the Input component. See Input documentation for available tokens._

## NumberInput API Reference

### `<NumberInput />`

| **Prop**                  | **Type**                                                                               | **Default** | **Description**                              |
| ------------------------- | -------------------------------------------------------------------------------------- | ----------- | -------------------------------------------- |
| **type**                  | string                                                                                 | _'number'_  | Input type (always 'number' for NumberInput) |
| **min**                   | number                                                                                 |             | Minimum value                                |
| **max**                   | number                                                                                 |             | Maximum value                                |
| **stepControl**           | number                                                                                 | _1_         | Step increment                               |
| **stepper**               | 'auto' \| 'chevron' \| 'separate'                                                      | _'auto'_    | Stepper type                                 |
| **unit**                  | string \| string[]                                                                     |             | Unit or list of units                        |
| **selectedUnit**          | string                                                                                 |             | Selected unit                                |
| **onUnitChange**          | (unit: string) => void                                                                 |             | Callback when unit changes                   |
| **allowInput**            | boolean                                                                                | _true_      | Allow manual input when using stepper        |
| **onValueChange**         | function (values, event?)                                                              |             | Callback when value changes (number format)  |
| **isClearable**           | boolean                                                                                | _false_     | Show clear button                            |
| **floatingLabel**         | React.ReactNode                                                                        |             | Floating label                               |
| **theme**                 | 'default' \| 'alternative'                                                             | _'default'_ | Theme                                        |
| **inputSize**             | 'sm' \| 'md'                                                                           | _'sm'_      | Input size                                   |
| **leftIcon**              | React.ReactNode                                                                        |             | Left icon                                    |
| **onClear**               | () => void                                                                             |             | Callback when clear                          |
| **...InputProps**         | [See Input](../base-input/dev)                                                         |             | Inherits all props from Input                |
| **...NumericFormatProps** | [See NumericFormat](https://s-yadav.github.io/react-number-format/docs/numeric_format) |             | Inherits all props from NumericFormat        |

> _Note:_ _NumberInput_ inherits all props from Input and the native input element. See [Input documentation](../base-input/dev) for full details on available props and tokens.
