## üß© Toast API Reference

### `<Toast />`

| **Prop**        | **Type**                       | **Default** | **Description**                              |
| --------------- | ------------------------------ | ----------- | -------------------------------------------- |
| **message**     | string \| ReactNode            |             | Toast message                                |
| **type**        | 'info' \| 'success' \| 'error' | _'info'_    | Toast type                                   |
| **duration**    | number                         | _3000_      | Duration in ms                               |
| **onClose**     | function ()                    |             | Callback when toast is closed                |
| **...divProps** | HTMLDivElement props           |             | All native div props (className, style, ...) |

> **_Note:_** _Toast_ **inherits** all native props from **HTMLDivElement**.
