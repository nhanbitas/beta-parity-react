## Anatomy <span id="anatomy"></span>

- **Container**: The outermost element with class `toast`.
- **Icon**: Optional icon or spinner, shown at the start (`toast-icon`).
- **Title**: Main heading of the toast (`toast-title`).
- **Message**: Main content/message (`toast-message`).
- **Action**: Optional action element (button or link, `toast-action`).
- **Close button**: Dismiss button (`toast-close`).
- **Progress bar**: Optional progress indicator (`toast-progress-bar`).

## States <span id="states"></span>

- **Default**: Toast is visible and enabled.
- **Hovered**: Mouse is over the toast (pauses auto-dismiss if enabled).
- **Pressed**: Close button or action is pressed.
- **Focused**: Toast is focused (keyboard navigation).
- **Disabled**: Not applicable (toasts are not disabled, but close button can be disabled).
- **Pending**: Shows spinner, disables close button.
- **Collapsed**: Used for stacking animation.
- **Expanded**: Default visible state.

## Options <span id="options"></span>

- **kind**: 'generic' | 'information' | 'affirmative' | 'cautionary' | 'adverse'
- **emphasis**: 'normal' | 'high'
- **height**: 'flexible' | 'compact'
- **position**: 'top-right' | 'top-center' | 'bottom-right' | 'bottom-center'
- **autoDismiss**: boolean
- **progressBar**: boolean
- **dismissButton**: boolean
- **importance**: number
- **pending**: boolean

## Behaviors <span id="behaviors"></span>

- **Auto-dismiss**: Toast can auto-dismiss after a duration, optionally paused on hover.
- **Stacking**: Multiple toasts can be stacked, with animation and scaling.
- **Progress bar**: Shows remaining time for auto-dismiss.
- **Keyboard accessibility**: Toast is focusable, close button is keyboard accessible.
- **Responsive**: Layout adapts to container width.

## Toast API Reference

### `<Toast />`

| **Prop**          | **Type**                                                                 | **Default** | **Description**                              |
| ----------------- | ------------------------------------------------------------------------ | ----------- | -------------------------------------------- |
| **title**         | string \| ReactNode                                                      |             | Toast title                                  |
| **message**       | string \| ReactNode                                                      |             | Toast message                                |
| **action**        | ReactNode                                                                |             | Optional action element                      |
| **kind**          | 'generic' \| 'information' \| 'affirmative' \| 'cautionary' \| 'adverse' | 'generic'   | Toast type                                   |
| **emphasis**      | 'normal' \| 'high'                                                       | 'high'      | Emphasis level                               |
| **height**        | 'flexible' \| 'compact'                                                  | 'flexible'  | Height variant                               |
| **position**      | 'top-right' \| 'top-center' \| 'bottom-right' \| 'bottom-center'         | 'top-right' | Toast position                               |
| **autoDismiss**   | boolean                                                                  | false       | Auto-dismiss after duration                  |
| **duration**      | number                                                                   | 5000        | Duration in ms                               |
| **pauseOnHover**  | boolean                                                                  | true        | Pause auto-dismiss on hover                  |
| **icon**          | ReactNode                                                                |             | Custom icon                                  |
| **progressBar**   | boolean                                                                  | false       | Show progress bar                            |
| **dismissButton** | boolean                                                                  | true        | Show close button                            |
| **importance**    | number                                                                   | 0           | Importance for stacking/sorting              |
| **pending**       | boolean                                                                  | false       | Pending state (shows spinner)                |
| **onDismissed**   | function ()                                                              |             | Callback when toast is dismissed             |
| **...divProps**   | HTMLDivElement props                                                     |             | All native div props (className, style, ...) |

> **_Note:_** _ToastOptions_ is imported from the Toast component and includes all above props except `id` and `removeToast` (which are internal).
