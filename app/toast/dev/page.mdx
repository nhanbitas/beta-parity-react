## Toast Container

- The `ToastContainer` is a wrapper component that manages the display of toast notifications. It can be configured with various options such as stacking, sorting by importance, and limiting the number of toasts displayed at once.
- Add the `ToastContainer` to your page/layout to enable toast notifications.

## Toast API Reference

### `<Toast />`

| **Prop**          | **Type**                                                                 | **Default**   | **Description**                              |
| ----------------- | ------------------------------------------------------------------------ | ------------- | -------------------------------------------- |
| **title**         | string \| ReactNode                                                      |               | Toast title                                  |
| **message**       | string \| ReactNode                                                      |               | Toast message                                |
| **action**        | ReactNode                                                                |               | Optional action element                      |
| **kind**          | 'generic' \| 'information' \| 'affirmative' \| 'cautionary' \| 'adverse' | _'generic'_   | Toast type                                   |
| **emphasis**      | 'normal' \| 'high'                                                       | _'high'_      | Emphasis level                               |
| **height**        | 'flexible' \| 'compact'                                                  | _'flexible'_  | Height variant                               |
| **position**      | 'top-right' \| 'top-center' \| 'bottom-right' \| 'bottom-center'         | _'top-right'_ | Toast position                               |
| **autoDismiss**   | boolean                                                                  | _false_       | Auto-dismiss after duration                  |
| **duration**      | number                                                                   | _5000_        | Duration in ms                               |
| **pauseOnHover**  | boolean                                                                  | _true_        | Pause auto-dismiss on hover                  |
| **icon**          | ReactNode                                                                |               | Custom icon                                  |
| **progressBar**   | boolean                                                                  | _false_       | Show progress bar                            |
| **dismissButton** | boolean                                                                  | _true_        | Show close button                            |
| **importance**    | number                                                                   | _0_           | Importance for stacking/sorting              |
| **pending**       | boolean                                                                  | _false_       | Pending state (shows spinner)                |
| **onDismissed**   | function ()                                                              |               | Callback when toast is dismissed             |
| **...divProps**   | HTMLDivElement props                                                     |               | All native div props (className, style, ...) |

> _Note: `ToastOptions` is imported from the Toast component and includes all above props except `id` and `removeToast` (which are internal)._
