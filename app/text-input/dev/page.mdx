## TextInput API Reference

### `<TextInput />`

| **Prop**          | **Type**                       | **Default** | **Description**                          |
| ----------------- | ------------------------------ | ----------- | ---------------------------------------- |
| **type**          | string                         | _'text'_    | Input type (always 'text' for TextInput) |
| **...InputProps** | [See Input](../base-input/dev) |             | Inherits all props from **Input**        |

> _Note: TextInput inherits all props from **Input** and the native **input** element. See [Input documentation](../base-input/dev) for full details on available props and tokens._
