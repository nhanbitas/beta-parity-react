## Table API Reference

### `<Table />`

| **Prop**          | **Type**               | **Default** | **Description**                                |
| ----------------- | ---------------------- | ----------- | ---------------------------------------------- |
| **data**          | array of objects       |             | Table data                                     |
| **columns**       | array of TableColumn   |             | Table columns                                  |
| **rowKey**        | string                 | _'id'_      | Unique key for each row                        |
| **size**          | 'sm' \| 'md' \| 'lg'   | _'md'_      | Table size                                     |
| **color**         | 'neutral' \| 'accent'  | _'neutral'_ | Color scheme                                   |
| **className**     | string                 |             | Custom className for container                 |
| **...tableProps** | HTMLTableElement props |             | All native table props (className, style, ...) |

> **_Note:_** Table expects each row to have a unique `id` field as the key. TableColumn type supports key, title, width, sortable, resizable, render, etc.
