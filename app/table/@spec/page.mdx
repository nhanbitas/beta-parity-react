## üß© Table API Reference

### `<Table />`

| **Prop**          | **Type**               | **Default** | **Description**                                |
| ----------------- | ---------------------- | ----------- | ---------------------------------------------- |
| **columns**       | array                  |             | Table columns                                  |
| **data**          | array                  |             | Table data rows                                |
| **rowKey**        | string \| function     |             | Unique row key                                 |
| **pagination**    | object                 |             | Pagination config                              |
| **loading**       | boolean                | _false_     | Loading state                                  |
| **onRowClick**    | function (row)         |             | Callback when row is clicked                   |
| **...tableProps** | HTMLTableElement props |             | All native table props (className, style, ...) |

> **_Note:_** _Table_ **inherits** all native props from **HTMLTableElement**.
