## Design Tokens for Slider

| **Token**                                              | **Color for**             | **State/Meaning** | **CSS Selector(s)**                                                                          |
| ------------------------------------------------------ | ------------------------- | ----------------- | -------------------------------------------------------------------------------------------- |
| **--par-color-border-slider-track-enabled**            | Slider track              | Default/Enabled   | `.slider-track`                                                                              |
| **--par-color-border-slider-progress-neutral-enabled** | Slider progress (neutral) | Default/Enabled   | `.slider-progress`                                                                           |
| **--par-color-border-slider-progress-accent-enabled**  | Slider progress (accent)  | Default/Enabled   | `.slider-progress.accent`                                                                    |
| **--par-color-border-slider-thumb-enabled**            | Slider thumb border       | Default/Enabled   | `input[type=range].par-input-slider::-webkit-slider-thumb`, `::-moz-range-thumb`             |
| **--par-color-bg-slider-thumb-enabled**                | Slider thumb background   | Default/Enabled   | `input[type=range].par-input-slider::-webkit-slider-thumb`, `::-moz-range-thumb`             |
| **--par-color-bg-slider-state-layer-hovered**          | Thumb hover state layer   | Hovered           | `input[type=range].par-input-slider:hover::-webkit-slider-thumb`, `::-moz-range-thumb:hover` |
| **--par-color-border-slider-discrete-neutral-active**  | Mark dot (in range)       | Active            | `.slider-dot.in-range`                                                                       |
| **--par-color-border-slider-discrete-neutral-enabled** | Mark dot (out range)      | Default           | `.slider-dot.out-range`                                                                      |
| **--par-color-text-primary**                           | Indicator text            | Any               | `.slider-thumb-indicator-text`                                                               |

> _Note: Slider uses the same design tokens as the Progress or Input component. See Progress/Input documentation for available tokens._

## Slider API Reference

### `<Slider />`

| **Prop**          | **Type**                   | **Default**     | **Description**                              |
| ----------------- | -------------------------- | --------------- | -------------------------------------------- |
| **mode**          | 'single' \| 'range'        | _'single'_      | Slider mode (single value or range)          |
| **min**           | number                     | _0_             | Minimum value                                |
| **max**           | number                     | _100_           | Maximum value                                |
| **step**          | number                     | _1_             | Step increment                               |
| **defaultValue**  | number \| [number, number] | _0_ \| _[0,50]_ | Initial value                                |
| **marks**         | Array of mark objects      | _[]_            | Mark points on the slider                    |
| **color**         | 'neutral' \| 'accent'      | _'neutral'_     | Slider color                                 |
| **orientation**   | 'horizontal' \| 'vertical' | _'horizontal'_  | Slider orientation                           |
| **indicator**     | 'normal' \| 'tooltip'      | _'normal'_      | Indicator type                               |
| **indicatorSide** | 'normal' \| 'reverse'      | _'normal'_      | Indicator position                           |
| **disabled**      | boolean                    | _false_         | Disable the slider                           |
| **onValueChange** | function (value)           |                 | Callback when value changes                  |
| **...divProps**   | HTMLDivElement props       |                 | All native div props (className, style, ...) |

> **_Note:_** _Slider_ **inherits** all native props from **HTMLDivElement**.
