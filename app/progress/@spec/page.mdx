## üß© Progress API Reference

### `<Progress />`

| **Prop**        | **Type**             | **Default** | **Description**                              |
| --------------- | -------------------- | ----------- | -------------------------------------------- |
| **value**       | number               |             | Current progress value                       |
| **max**         | number               | _100_       | Maximum value                                |
| **min**         | number               | _0_         | Minimum value                                |
| **label**       | string \| ReactNode  |             | Optional label                               |
| **showLabel**   | boolean              | _false_     | Show label                                   |
| **size**        | 'sm' \| 'md' \| 'lg' | _'md'_      | Progress bar size                            |
| **color**       | string               |             | Progress bar color                           |
| **...divProps** | HTMLDivElement props |             | All native div props (className, style, ...) |

> **_Note:_** _Progress_ **inherits** all native props from **HTMLDivElement**.
