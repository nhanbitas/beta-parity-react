## Anatomy <span id="anatomy"></span>

- **Radio**: The main input element (`input.par-radio`)
- **Radio icon**: Visual indicator (`.radio-icon`)
- **Label**: Main label (`.input-label`)
- **Sublabel**: Optional sublabel (`.input-sublabel`)
- **Wrapper**: Container (`.radio-wrapper`)
- **Group**: For grouped radios (`.radio-group`)

## States <span id="states"></span>

- **Default**: Unchecked, enabled
- **Checked**: Selected state
- **Hovered**: Mouse over
- **Focused**: Keyboard or mouse focus
- **Disabled**: Not interactive
- **Error**: Invalid/error state

## Options <span id="options"></span>

- **color**: 'neutral' | 'accent'
- **label**: string | ReactNode
- **sublabel**: string | ReactNode
- **disabled**: boolean
- **radioWrapperProps**: any
- **All InputProps**: Inherits all props from Input

## Behaviors <span id="behaviors"></span>

- **Keyboard**: Fully accessible via Tab, Space/Enter to toggle
- **Group**: Only one radio in a group can be selected
- **Focus ring**: Visible on keyboard focus
- **Animation**: SVG icon animates on check/uncheck
- **ARIA**: Uses appropriate ARIA attributes for accessibility

## üß© Radio API Reference

### `<Radio />`

| **Prop**              | **Type**                  | **Default** | **Description**                   |
| --------------------- | ------------------------- | ----------- | --------------------------------- |
| **checked**           | boolean                   |             | Checked state                     |
| **onChange**          | function (event)          |             | Callback when value changes       |
| **value**             | string \| number          |             | Value of the radio                |
| **label**             | string \| ReactNode       |             | Label for the radio               |
| **sublabel**          | string \| ReactNode       |             | Sublabel for the radio            |
| **disabled**          | boolean                   | _false_     | Disable the radio                 |
| **color**             | string                    |             | Radio color                       |
| **radioWrapperProps** | any                       |             | Props for the radio wrapper       |
| **...InputProps**     | [See Input](../BaseInput) |             | Inherits all props from **Input** |

> **_Note:_** _Radio_ **inherits** all props from **Input** and the native **input** element. See [Input documentation](../BaseInput) for full details on available props and tokens.
