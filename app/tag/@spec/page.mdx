## Anatomy

- **Container**: The outermost encompassing element.
- **Label**: The text that describes the tag‚Äôs content.
- **Icon**: An optional visual element that accompanies the tag‚Äôs content.
- **Avatar**: An optional visual representation of a user or entity associated with the tag.
- **Remove button**: An optional button that removes the tag from the interface.

## Keyboard interactions

| Key                                    | Focus              | Interaction                                                                                                                                                                                       |
| -------------------------------------- | ------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| <kbd>Tab</kbd>                         | Focusable elements | Moves focus in the forward sequence, as follows: previous external element ‚Üí container ‚Üí remove button ‚Üí next external element.<br></br>Non-focusable elements are skipped in the above sequence. |
| <kbd>Shift</kbd>+<kbd>Tab</kbd>        | Focusable elements | Moves focus in the reverse sequence.                                                                                                                                                              |
| <kbd>Del</kbd> or <kbd>Backspace</kbd> | Container          | Removes the tag from the user interface.                                                                                                                                                          |
| <kbd>Space</kbd> or <kbd>Enter</kbd>   | Remove button      | Removes the tag from the user interface.                                                                                                                                                          |
