## Design Tokens for Tag

| **Token**                              | **Color for**            | **State/Meaning** | **CSS Selector(s)**            |
| -------------------------------------- | ------------------------ | ----------------- | ------------------------------ |
| `--par-color-bg-tag`                   | Tag background           | Default/Enabled   | `.tag`                         |
| `--par-color-bg-tag-hovered`           | Tag background           | Hovered           | `.tag:hover`                   |
| `--par-color-bg-tag-selected`          | Tag background           | Selected          | `.tag.selected`                |
| `--par-color-bg-tag-disabled`          | Tag background           | Disabled          | `.tag:disabled`                |
| `--par-color-border-tag`               | Tag border               | Any               | `.tag`                         |
| `--par-color-text-tag`                 | Tag text                 | Default/Enabled   | `.tag`                         |
| `--par-color-text-tag-disabled`        | Tag text                 | Disabled          | `.tag:disabled`                |
| `--par-color-icon-tag`                 | Tag icon                 | Any               | `.tag .tag-icon`               |
| `--par-color-bg-tag-remove-hovered`    | Remove button background | Hovered           | `.tag .tag-close-btn:hover`    |
| `--par-color-bg-tag-remove-pressed`    | Remove button background | Pressed           | `.tag .tag-close-btn:active`   |
| `--par-color-bg-tag-remove-disabled`   | Remove button background | Disabled          | `.tag .tag-close-btn:disabled` |
| `--par-color-text-tag-remove`          | Remove button text/icon  | Any               | `.tag .tag-close-btn`          |
| `--par-color-text-tag-remove-disabled` | Remove button text/icon  | Disabled          | `.tag .tag-close-btn:disabled` |

> _Note: Tag uses the same design tokens as the Badge or Chip component. See Badge/Chip documentation for available tokens._

## Tag API Reference

### `<Tag />`

| **Prop**         | **Type**              | **Default** | **Description**                               |
| ---------------- | --------------------- | ----------- | --------------------------------------------- |
| **label**        | string \| ReactNode   |             | Tag label                                     |
| **color**        | string                |             | Tag color                                     |
| **size**         | 'sm' \| 'md' \| 'lg'  | _'md'_      | Tag size                                      |
| **closable**     | boolean               | _false_     | Show close button                             |
| **onClose**      | function ()           |             | Callback when close is clicked                |
| **...spanProps** | HTMLSpanElement props |             | All native span props (className, style, ...) |

> **_Note:_** _Tag_ **inherits** all native props from **HTMLSpanElement**.
