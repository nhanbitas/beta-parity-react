## Design Tokens for Tabs

| **Token**                                      | **Color for**  | **State/Meaning**         | **CSS Selector(s)**    |
| ---------------------------------------------- | -------------- | ------------------------- | ---------------------- |
| **`--par-color-bg-tab-item-enabled`**          | Tab background | Default/Enabled           | `.tab-button`          |
| **`--par-color-bg-tab-item-hovered`**          | Tab background | Hovered                   | `.tab-button:hover`    |
| **`--par-color-bg-tab-item-neutral-active`**   | Tab background | Active/Selected (neutral) | `.tab-button.active`   |
| **`--par-color-bg-tab-item-accent-active`**    | Tab background | Active/Selected (accent)  | `.tab-button.active`   |
| **`--par-color-bg-tab-item-disabled`**         | Tab background | Disabled                  | `.tab-button:disabled` |
| **`--par-color-border-tab-container`**         | Tabs border    | Any                       | `.tabs-nav-wrapper`    |
| **`--par-color-text-tab-item-enabled`**        | Tab text       | Default/Enabled           | `.tab-button`          |
| **`--par-color-text-tab-item-hovered`**        | Tab text       | Hovered                   | `.tab-button:hover`    |
| **`--par-color-text-tab-item-neutral-active`** | Tab text       | Active/Selected (neutral) | `.tab-button.active`   |
| **`--par-color-text-tab-item-accent-active`**  | Tab text       | Active/Selected (accent)  | `.tab-button.active`   |
| **`--par-color-text-tab-item-disabled`**       | Tab text       | Disabled                  | `.tab-button:disabled` |

> _Note: Tabs uses the same design tokens as the Button or Navigation component. See Button/Navigation documentation for available tokens._

## Tabs API Reference

### `<Tabs />`

| **Prop**          | **Type**                                       | **Default**                 | **Description**                              |
| ----------------- | ---------------------------------------------- | --------------------------- | -------------------------------------------- |
| **data**          | array of TabItemProps                          |                             | List of tab items                            |
| **size**          | 'sm' \| 'md'                                   | _'md'_                      | Tabs size                                    |
| **color**         | 'neutral' \| 'accent'                          | _'neutral'_                 | Tabs color                                   |
| **theme**         | 'default' \| 'alternative'                     | _'default'_                 | Tabs theme                                   |
| **side**          | 'left' \| 'right' \| 'top' \| 'bottom'         | _'left'_                    | Tabs position                                |
| **flipped**       | boolean                                        | _false_                     | Flip tab border position                     |
| **indicatorSide** | 'same' \| 'opposite'                           | _'same'_                    | Indicator side relative to border            |
| **navProps**      | HTMLDivElement props                           |                             | Props for nav container                      |
| **bodyProps**     | HTMLDivElement props                           |                             | Props for body container                     |
| **scrollValue**   | object (`{ value: number, duration: number }`) | `{ value: 4, duration: 1 }` | Scroll config for nav                        |
| **...divProps**   | HTMLDivElement props                           |                             | All native div props (className, style, ...) |
