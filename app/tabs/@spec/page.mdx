## üß© Tabs API Reference

### `<Tabs />`

| **Prop**        | **Type**                               | **Default** | **Description**                              |
| --------------- | -------------------------------------- | ----------- | -------------------------------------------- |
| **tabs**        | array                                  |             | List of tab items                            |
| **activeKey**   | string \| number                       |             | Active tab key                               |
| **onChange**    | function (key)                         |             | Callback when tab changes                    |
| **position**    | 'top' \| 'bottom' \| 'left' \| 'right' | _'top'_     | Tabs position                                |
| **size**        | 'sm' \| 'md' \| 'lg'                   | _'md'_      | Tabs size                                    |
| **...divProps** | HTMLDivElement props                   |             | All native div props (className, style, ...) |

> **_Note:_** _Tabs_ **inherits** all native props from **HTMLDivElement**.
