## Design Tokens for Avatar

| **Token**                                 | **Color for**        | **State/Meaning**         | **CSS Selector(s)**                                                |
| ----------------------------------------- | -------------------- | ------------------------- | ------------------------------------------------------------------ |
| **--par-color-bg-avatar**                 | Avatar background    | Default background        | `.avatar`, `.avatar-group-overflow`                                |
| **--par-color-bg-avatar-disabled**        | Avatar background    | Disabled state            | `.avatar.disabled`                                                 |
| **--par-color-bg-avatar-trigger**         | AvatarTrigger bg     | Avatar trigger background | `.avatar-trigger`                                                  |
| **--par-color-border-avatar**             | Avatar border        | Default border            | `.avatar`, `.avatar.border`, `.avatar-group-item`                  |
| **--par-color-border-avatar-alternative** | Avatar border alt    | Alternative border style  | `.avatar.border-alternative`                                       |
| **--par-color-border-avatar-gray**        | Avatar border gray   | Gray status border        | `.avatar.status-border[data-status-color="gray"]`                  |
| **--par-color-border-avatar-red**         | Avatar border red    | Red status border         | `.avatar.status-border[data-status-color="red"]`                   |
| **--par-color-border-avatar-green**       | Avatar border green  | Green status border       | `.avatar.status-border[data-status-color="green"]`                 |
| **--par-color-border-avatar-yellow**      | Avatar border yellow | Yellow status border      | `.avatar.status-border[data-status-color="yellow"]`                |
| **--par-color-text-avatar**               | Initials/placeholder | Initials & placeholder    | `.avatar-initials`, `.avatar-placeholder`, `.avatar-overflow-text` |
| **--par-color-text-avatar-disabled**      | Avatar text          | Disabled state            | `.avatar.disabled`, `.avatar.disabled .avatar-img`                 |
| **--par-color-text-avatar-trigger**       | AvatarTrigger text   | Avatar trigger text       |                                                                    |
| **--avatar-group-spacing**                | Group spacing        | Spacing between avatars   | `.avatar-group.direction-row`, `.avatar-group.direction-column`    |

> _Note: Status border tokens are set via `data-status-color` and mapped to the appropriate color variable. Supported colors: gray, red, green, yellow._

## Avatar API Reference

### `<Avatar />`

| **Prop**        | **Type**                                      | **Default** | **Description**                              |
| --------------- | --------------------------------------------- | ----------- | -------------------------------------------- |
| **src**         | string                                        |             | Image URL                                    |
| **initials**    | string                                        |             | Initials fallback                            |
| **size**        | '2xs' \| 'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl' | _'md'_      | Avatar size                                  |
| **status**      | 'online' \| 'offline' \| 'busy' \| 'away'     |             | Status indicator                             |
| **borderStyle** | 'default' \| 'alternative' \| 'none'          | _'default'_ | Border style                                 |
| **children**    | ReactNode                                     |             | Custom content                               |
| **...divProps** | HTMLDivElement props                          |             | All native div props (className, style, ...) |

### `<AvatarGroup />`

| **Prop**            | **Type**                                | **Default** | **Description**                              |
| ------------------- | --------------------------------------- | ----------- | -------------------------------------------- |
| **max**             | number                                  | _5_         | Max avatars to display before overflow.      |
| **direction**       | 'row' \| 'column'                       | _'row'_     | Stack direction.                             |
| **spacing**         | string                                  | _'-0.5rem'_ | Spacing between avatars.                     |
| **size**            | AvatarSize                              | _'medium'_  | Size for all avatars in the group.           |
| **borderStyle**     | 'default' \| 'alternative' \| 'none'    | _'default'_ | Border style for all avatars.                |
| **renderOverflow**  | (overflowCount: number) => ReactNode    |             | Custom overflow indicator.                   |
| **onOverflowClick** | (e: MouseEvent[HTMLDivElement]) => void |             | Handler for overflow avatar click.           |
| **...divProps**     | HTMLDivElement props                    |             | All native div props (className, style, ...) |

### `<AvatarTrigger />`

| **Prop**           | **Type**                                | **Default** | **Description**                         |
| ------------------ | --------------------------------------- | ----------- | --------------------------------------- |
| **onClick**        | (e: MouseEvent[HTMLDivElement]) => void |             | Click handler for the trigger.          |
| **active**         | boolean                                 | _false_     | Whether the trigger is active/selected. |
| **...AvatarProps** | [See above](#avatar)                    |             | All props from **Avatar**.              |

> **Note:** _AvatarTrigger_ **extends** all props from **Avatar**. See the Avatar props table above for details.
