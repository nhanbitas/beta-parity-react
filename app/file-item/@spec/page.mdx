## FileItem API Reference

### `<FileItem />`

| **Prop**        | **Type**             | **Default** | **Description**                                 |
| --------------- | -------------------- | ----------- | ----------------------------------------------- |
| **file**        | File \| object       |             | File object or file data to display             |
| **onRemove**    | function (file)      |             | Callback when remove is clicked                 |
| **onDownload**  | function (file)      |             | Callback when download is clicked               |
| **progress**    | number               |             | Upload/download progress (0-100)                |
| **status**      | string               |             | File status (e.g. 'uploading', 'done', 'error') |
| **...divProps** | HTMLDivElement props |             | All native div props (className, style, ...)    |

> **_Note:_** _FileItem_ **inherits** all native props from **HTMLDivElement**.
