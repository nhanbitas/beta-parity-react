## üß© Textarea API Reference

### `<Textarea />`

| **Prop**          | **Type**                  | **Default** | **Description**                   |
| ----------------- | ------------------------- | ----------- | --------------------------------- |
| **onChange**      | function (event)          |             | Callback when value changes       |
| **value**         | string                    |             | Controlled value                  |
| **placeholder**   | string                    |             | Placeholder text                  |
| **rows**          | number                    |             | Number of visible text lines      |
| **...InputProps** | [See Input](../BaseInput) |             | Inherits all props from **Input** |

> **_Note:_** _Textarea_ **inherits** all props from **Input** and the native **textarea** element. See [Input documentation](../BaseInput) for full details on available props and tokens.
