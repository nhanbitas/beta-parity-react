## InlineLink API Reference

### `<InlineLink />`

| **Prop**      | **Type**                | **Default** | **Description**                                 |
| ------------- | ----------------------- | ----------- | ----------------------------------------------- |
| **href**      | string                  |             | Link URL                                        |
| **target**    | string                  |             | Target attribute (\_blank, \_self, etc.)        |
| **size**      | 'sm' \| 'md' \| 'lg'    | _'md'_      | Link size                                       |
| **color**     | string                  |             | Link color                                      |
| **disabled**  | boolean                 | _false_     | Disable the link                                |
| **underline** | boolean                 | _true_      | Show underline                                  |
| **icon**      | ReactNode               |             | Icon before/after text                          |
| **iconOnly**  | boolean                 | _false_     | Icon only, no text                              |
| **...aProps** | HTMLAnchorElement props |             | All native anchor props (className, style, ...) |

> _Note:_ _InlineLink_ inherits all native props from HTMLAnchorElement.
