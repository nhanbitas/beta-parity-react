## Anatomy <span id="anatomy"></span>

- **Container**: The outermost `div` with class `corner-indicator-container` (plus position modifier classes: `top-right`, `top-left`, `bottom-right`, `bottom-left`).
- **Badge**: The badge indicator (when `label` is provided), rendered as a child with class `badge`.
- **Dot**: The dot indicator (when `label` is not provided), rendered as a child with class `dot`.
- **Outline**: Optional ring outline, applied via `bordered` class and CSS token.
- **Children**: The wrapped content (e.g., button, avatar, etc.).

## States <span id="states"></span>

- **Default**: Indicator is visible in the specified position.
- **Outlined**: Outline ring is shown when `outline` prop is true.
- **Pulse**: Dot indicator animates when `pulse` prop is true.
- **Disabled**: If `disable` is true, the indicator is not rendered (children only).

## Options <span id="options"></span>

- **label**: string ‚Äî Text inside the badge indicator.
- **icon**: ReactNode ‚Äî Icon for badge indicator.
- **outline**: boolean ‚Äî Show outline ring.
- **pulse**: boolean ‚Äî Pulse animation (dot only).
- **offset, xOffset, yOffset**: number ‚Äî Control indicator position offset (px).
- **disable**: boolean ‚Äî Hide indicator, show only children.
- **position**: 'top-left' | 'top-right' | 'bottom-left' | 'bottom-right' ‚Äî Indicator position.
- **color**: 'gray' | 'orange' | 'violet' | 'green' | 'red' | 'yellow' | 'blue' | 'lime' | 'cyan' ‚Äî Indicator color.
- **size**: 'md' | 'sm' | 'xs' | 'lg' ‚Äî Indicator size.
- **variant**: 'solid' | 'outlined' | 'glass' ‚Äî Visual style (badge only).
- **growDirection**: 'symmetric' | 'inward' | 'outward' ‚Äî Direction the indicator grows.
- **indicatorProps**: object ‚Äî Extra props for Badge/Dot (except size, color, variant, pulse).
- **...divProps**: All native div props (className, style, ...).

## Behaviors <span id="behaviors"></span>

- Renders a badge or dot indicator in the specified corner of the container.
- Supports outline ring via CSS token and `outline` prop.
- Supports pulse animation for dot indicator.
- Supports custom offset and grow direction for flexible positioning.
- If `disable` is true, only children are rendered (no indicator).
- Responsive to light/dark color schemes via CSS tokens.
