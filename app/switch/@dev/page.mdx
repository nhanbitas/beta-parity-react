## Design Tokens for Switch

| **Token**                                               | **Color for**           | **State/Meaning**  | **CSS Selector(s)**                                  |
| ------------------------------------------------------- | ----------------------- | ------------------ | ---------------------------------------------------- |
| **--par-color-bg-switch-target-handle**                 | Switch thumb background | Enabled            | `.switch-target`                                     |
| **--par-color-bg-switch-target-handle-disabled**        | Switch thumb background | Disabled           | `.switch-target`                                     |
| **--par-color-bg-switch-target-track-uncheck**          | Switch track background | Unchecked          | `.switch`                                            |
| **--par-color-bg-switch-target-track-uncheck-disabled** | Switch track background | Unchecked Disabled | `.switch:disabled`                                   |
| **--par-color-bg-switch-target-track-check**            | Switch track background | Checked            | `.switch[aria-checked='true']`                       |
| **--par-color-bg-switch-target-track-check-disabled**   | Switch track background | Checked Disabled   | `.switch:disabled[aria-checked='true']`              |
| **--par-color-border-switch-unchecked**                 | Switch border           | Unchecked          | `.switch`                                            |
| **--par-color-border-switch-checked**                   | Switch border           | Checked            | `.switch[aria-checked='true']`                       |
| **--par-color-border-switch-unchecked-disabled**        | Switch border           | Unchecked Disabled | `.switch:disabled`                                   |
| **--par-color-border-switch-checked-disabled**          | Switch border           | Checked Disabled   | `.switch:disabled[aria-checked='true']`              |
| **--par-color-text-switch-target-unchecked-enabled**    | Switch icon             | Unchecked          | `.switch-icon`                                       |
| **--par-color-text-switch-target-checked-enabled**      | Switch icon             | Checked            | `.switch-icon`                                       |
| **--par-color-text-switch-target-unchecked-disabled**   | Switch icon             | Unchecked Disabled | `.switch:disabled .switch-icon`                      |
| **--par-color-text-switch-target-checked-disabled**     | Switch icon             | Checked Disabled   | `.switch:disabled[aria-checked='true'] .switch-icon` |

> _Note: Switch uses design tokens defined in `registry/src/components/Switch/variables.css`. See Input/Toggle documentation for additional shared tokens._

## Switch API Reference

### `<Switch />`

| **Prop**           | **Type**                                                                               | **Default** | **Description**                           |
| ------------------ | -------------------------------------------------------------------------------------- | ----------- | ----------------------------------------- |
| **active**         | boolean                                                                                |             | Controlled state                          |
| **defaultActive**  | boolean                                                                                | _false_     | Uncontrolled initial state                |
| **switchSize**     | 'sm' \| 'md' \| 'lg'                                                                   | _'md'_      | Switch size                               |
| **disabled**       | boolean                                                                                | _false_     | Disable the switch                        |
| **icon**           | boolean                                                                                | _false_     | Show icon on thumb                        |
| **onToggle**       | function (active: boolean)                                                             |             | Callback when value changes               |
| **...buttonProps** | [See MDN](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/button#attributes) |             | Inherits all props from native **button** |

> **_Note:_** _Switch_ **does not inherit** Input props. See [Button documentation](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/button#attributes) for full details on available props.
