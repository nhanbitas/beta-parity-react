## Checkbox API Reference

### `<Checkbox />`

| **Prop**                 | **Type**                 | **Default** | **Description**                   |
| ------------------------ | ------------------------ | ----------- | --------------------------------- |
| **color**                | 'neutral' \| 'accent'    | _'neutral'_ | Checkbox color                    |
| **label**                | string \| ReactNode      |             | Main label                        |
| **sublabel**             | string \| ReactNode      |             | Sublabel                          |
| **indeterminate**        | boolean                  | _false_     | Indeterminate state               |
| **checkboxWrapperProps** | object                   |             | Props for outer wrapper           |
| **...InputProps**        | [See below](#inputprops) |             | Inherits all props from **Input** |

---

### `<CheckboxGroup />`

| **Prop**         | **Type**                         | **Default** | **Description**                         |
| ---------------- | -------------------------------- | ----------- | --------------------------------------- |
| **tree**         | DataItems (array)                |             | Tree data for group                     |
| **children**     | ReactNode                        |             | Children nodes                          |
| **onChange**     | function (value, tree, getGroup) |             | Callback when value changes             |
| **onParse**      | function (value)                 |             | Callback when parsing data              |
| **label**        | string \| ReactNode              |             | Group label                             |
| **sublabel**     | string \| ReactNode              |             | Group sublabel                          |
| **value**        | string                           |             | Group value                             |
| **checked**      | boolean                          |             | Checked state for group                 |
| **color**        | 'neutral' \| 'accent'            | _'neutral'_ | Group color                             |
| **...BaseProps** | [See Base](../Base)              |             | Inherits all layout props from **Base** |

---

### <span id="inputprops"></span>**InputProps** _(inherited from input element)_

| **Prop**                      | **Type**                   | **Description**                                                            |
| ----------------------------- | -------------------------- | -------------------------------------------------------------------------- |
| **wrapperProps**              | object                     | Props for input wrapper                                                    |
| **clearBtnProps**             | object                     | Props for clear button                                                     |
| **isClearable**               | boolean                    | Show clear button                                                          |
| **floatingLabel**             | ReactNode                  | Floating label                                                             |
| **onClear**                   | () => void                 | Callback when clear is clicked                                             |
| **ActionBtn**                 | ReactNode                  | Action button on the right                                                 |
| **isError**                   | boolean                    | Error state                                                                |
| **errorMessage**              | string                     | Error message                                                              |
| **leftIcon**                  | ReactNode                  | Left icon                                                                  |
| **theme**                     | 'default' \| 'alternative' | Theme                                                                      |
| **inputSize**                 | 'sm' \| 'md'               | Input size                                                                 |
| ...and all native input props |                            | [See MDN](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/input) |

> **_Note:_** _Checkbox_ **inherits** all props from **Input** and **input element**.
